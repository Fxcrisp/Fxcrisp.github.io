Markdown语言学习（一）

发表于2023-05-28|更新于2023-06-08|博客基础markdown
|字数总计:403|阅读时长:1分钟|阅读量:

Markdown 是一种用来写作的轻量级「标记语言」，它用简洁的语法代替排版。

我将参考Markdown 的官方文档和认识与入门 Markdown来进行学习，在简书上进行文章的实时预览。

标题

# 一级标题
## 二级标题
### 三级标题
以此类推，总共六级标题，建议在井号后加一个空格，这是最标准的 Markdown 语法。

序号

列表的显示只需要在文字前加上 - 或 * 即可变为无序列表，有序列表则直接在文字前加 1.2.3. 符号要和文字之间加上一个字符的空格。

引用

只需要在文本前加入 > 这种尖括号（大于号）即可。

图片

![文字](图片的url地址)

链接

[文字](url地址)

字体的效果

  • 粗体：两个 * 或 _ 包含一段文本
  • 斜体：一个 * 或 _ 包含一段文本
  • 标记文本（高光）：两个==包含一段文本
  • 删除文本：两个 ~~ 包含一段文本

表格

| Tables | Are | Cool |
| ------------- |:-------------:| ------:|
| col 3 is | right-aligned | $1600 |
这段代码实际效果如下：

Tables Are Cool
col 3 is right-aligned $1600
  • 对齐
    :— 表示左对齐，:—: 表示居中，—:表示右对齐。
  • 换行
    在表格内换行，可以使用 html 的 br 标签。
    <br>

代码框

  • 一句：用两个“ ` ”把中间的代码包裹起来
  • 一段：
\````代码类型
代码块
\````
  • 使用 tab 键缩进
  • 被md语法解析了的特殊字符，可在前面加 “ / ”

分割线

分割线的语法只需要另起一行，连续输入三个星号 *** 即可。
